# Лабораторная работа №9: Анализ шума

(загрузил без .wav файла, потому что он весил слишком много и не вмещался в лимиты GitHub. по факту взял видео https://www.youtube.com/watch?v=HIe6Kaj4V7Q и сконвертил его в .wav)

## Что сделано
1. Загружается подготовленная запись `sweden.wav`
2. Учитывается число каналов: исходный файл стерео, для анализа строится моно-сигнал как среднее каналов, а очищенный звук сохраняется снова в стерео
3. Строятся спектрограммы до и после обработки
4. Шум оценивается по участкам с малой энергией
5. Выполняется шумопонижение методом спектрального вычитания.
6. Восстанавливается звуковая дорожка `sweden_denoised.wav`
7. Находятся области максимальной энергии при `delta t = 0.1 c` и `delta f = 50 Гц`

## Формулы и параметры
Для спектрограммы используется оконное преобразование Фурье:

```text
STFT{x[n]}(m, w) = sum x[n]w[n - m]e^(-jwn)
spectrogram{x[n]} = |STFT{x[n]}|^2
```

Окно Ханна:

```text
w(n) = 0.5 - 0.5 * cos(2*pi*n / (N - 1))
```

Параметры:

```text
длина окна = 50 мс
перекрытие = 75%
шкала частот на спектрограмме = логарифмическая
```

Метод спектрального вычитания:

```text
Y[f, t] = max(X[f, t] - kW[f, t], 0)
```

где:
- `X[f, t]` — амплитудный спектр исходного сигнала
- `W[f, t]` — оценка амплитудного спектра шума
- `Y[f, t]` — амплитудный спектр после очистки
- фаза берется от исходного сигнала
- затем выполняется обратное STFT

В программе:

```text
k = 1.6
нижняя граница усиления = 0.08
```

Нижняя граница усиления оставляет небольшую часть шума и уменьшает эффект "музыкального шума"

## Структура

- `sweden.wav` — исходная запись
- `output/spectrograms/sweden_before.png` — спектрограмма до обработки
- `output/spectrograms/sweden_after.png` — спектрограмма после обработки
- `output/audio/sweden_denoised.wav` — восстановленная звуковая дорожка
- `output/tables/energy_maxima.csv` — таблица максимумов энергии
- `output/reports/report.txt` — краткий отчет

## Запуск
Из директории `lab9`:

```bash
python main.py
```

С явным указанием входного файла и папки результатов:

```bash
python main.py --input ./sweden.wav --output-dir ./output
```
