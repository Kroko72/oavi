# Лабораторная работа №7: Классификация символов на основе признаков

## Вариант
- Алфавит: `греческие строчные буквы`
- Шрифт эталонов: `Times New Roman`
- Исходный размер шрифта: `52`
- Экспериментальный размер шрифта: `58`
- Исходная строка: `σε αγαπω`
- Дополнительная длинная строка: `σε αγαπω πολυ και σε θελω`

Используется набор символов:

```text
αβγδεζηθικλμνξοπρστυφχψω
```

## Что сделано
1. Сгенерированы эталоны всех символов выбранного алфавита.
2. Для каждого эталона рассчитаны нормированные признаки:
   - относительная масса;
   - относительные координаты центра тяжести;
   - нормированные осевые моменты инерции `Ix` и `Iy`.
3. Для входной строки построены вертикальный и горизонтальный профили.
4. По профилям выполнена сегментация строки на отдельные символы.
5. Для каждого найденного символа рассчитаны те же признаки.
6. Для каждого символа строки посчитаны расстояния до всех эталонов по евклидовой метрике.
7. Гипотезы отсортированы по убыванию меры близости и сохранены в файл.
8. Построена распознанная строка по первой гипотезе каждого символа.
9. Посчитаны количество ошибок и процент верно распознанных символов.
10. Проведен эксперимент с тем же текстом, но с размером шрифта `58`.
11. Добавлен дополнительный прогон на более длинной фразе и на ее варианте с другим размером шрифта.

## Формулы
Признаки считаются так:

- `weight_rel = weight / (M * N)`
- `x_rel = x / (M - 1)`
- `y_rel = y / (N - 1)`
- `Ix_rel = Ix / (M^2 * N^2)`
- `Iy_rel = Iy / (M^2 * N^2)`

Расстояние между символами:

```text
d = sqrt(sum((f1 - f2)^2))
```

Так как все признаки нормированы, мера близости берется как:

```text
s = 1 - d / sqrt(n)
```

При `d = 0` получаем `s = 1`.

## Основные результаты
После прогона для основной фразы получено:

- исходная строка без пробелов: `σεαγαπω`
- распознанная исходная строка: `σεογαπω`
- ошибок: `1`
- точность: `85.71%`

Эксперимент со шрифтом `58`:

- распознанная строка: `σεογαπω`
- ошибок: `1`
- точность: `85.71%`

Для дополнительной длинной фразы `σε αγαπω πολυ και σε θελω`:

- строка без пробелов: `σεαγαπωπολυκαισεθελω`
- распознанная строка при размере `52`: `σεογαπωπολωκαεσεθελω`
- ошибок: `3`
- точность: `85.00%`

Для той же длинной фразы при размере `58`:

- распознанная строка: `σεογαπωπολοκαεσεθελω`
- ошибок: `3`
- точность: `85.00%`

Классификатор по-прежнему чаще всего путает близкие по глобальным признакам символы вроде `α` и `ο`, `υ` и `ω`, `ι` и `ε`.
